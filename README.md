## :bookmark: About
  
Web application with calculation of the Notion Promoter Score.

## :rocket: Technologies

-   [Node.js](https://nodejs.org/en/)
-   [Express](https://expressjs.com/)
-   [Nodemailer](https://nodemailer.com/about/)

## :boom: How to run

-   ### **Prerequisites**

    -  	It is **required** to have **[Node.js](https://nodejs.org/en/)** installed on the computer
    -   It is **required** to have **[Git](https://git-scm.com/)** installed and configured on the computer
    -  	Also, it is **must** have a package manager like **[NPM](https://www.npmjs.com/)**.

1. Clone the repository:

```sh
  $ git clone https://github.com/thallesyasmim/Real-time-Chat.git
```

2. Running the application:

```sh
  # Server
  # Install project dependencies
  $ npm install
  # Get started server
  $ npm run dev
```
